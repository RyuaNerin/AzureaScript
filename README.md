# ExtensionTweet.js

- `W` : 수동 RT
- `E` : 모두에게 멘션
- `R` : 한명에게 멘션
- `Ctrl` + `D` : 트윗 삭제

# ExtensionView.js

- `Ctrl`/`Shift` + `R`/`G`/`A`/`B`

 |단축키|설명|
|:-:|---|
|`Ctrl`|나의|
|`Shift`|트윗의 작성자|
|`R`|팔로워 목록|
|`G`|팔로잉 목록|
|`A`|관심글 목록|
|`B`|유저 정보|
